# 📊 Loan Default Risk Analysis Dashboard

## 🔍 Overview

This project is a data visualization dashboard created using Microsoft Power BI to analyze loan default risk.
Python was used for data cleaning and merging multiple datasets before building the dashboard.

---

## 🎯 Objective

* Analyze customer data to identify default risk
* Compare default rates across different risk categories
* Understand how age and other factors impact loan defaults

---

## 🛠 Tools & Technologies Used

* Microsoft Power BI (Dashboard & Visualization)
* Python (Data Cleaning & Data Merging)
* Pandas & NumPy (Data Processing)

---

## ⚙️ Data Preparation (Python Work)

* Cleaned missing and inconsistent values
* Merged multiple datasets into a single structured dataset
* Performed data transformation for better analysis
* Removed duplicates and handled incorrect data entries

---

## 📌 Key Insights

* Total Customers: 308K

* Total Defaulters: 25K

* Overall Default Rate: 8.07%

* High Risk customers have the highest default rate (~17%)

* Medium Risk customers show moderate default (~8%)

* Low Risk customers have very low default (~3–4%)

* Certain age groups show higher default patterns

---

## 📊 Dashboard Features

* Risk Category-wise customer distribution
* Default Rate gauge visualization
* Default Rate by Age analysis
* Comparison of default rate across risk categories
* Interactive filters (Risk Category, Gender)

---

## 📁 Dataset

* Source: Home Credit Loan Dataset
* Includes customer details, risk categories, and default information

---

## 🚀 Conclusion

This dashboard helps in identifying risky customers and supports better decision-making for loan approvals.
It can be useful for banks and financial institutions to reduce financial risk.
